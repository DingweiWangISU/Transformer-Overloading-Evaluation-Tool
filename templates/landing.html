The EPRC Tranformer Overloading Evaluation Tool can be used to determine which transformers are likely to overload given your existing infrastructure and estimated heat pump (HP) and electric vehcile (EV) penetration.

Two files are required to use this tool: AMI-Data.xlsx and TFC-Info.xlsx.

  • The AMI-Data.xlsx data file contains one year of hourly AMI data for all customers.
  •  
  • The TFC-Info.xlsx data file contains transformer specifications and transformer-customer connectivity.
